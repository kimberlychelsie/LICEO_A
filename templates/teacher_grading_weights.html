Grading weights are automatically set per subject based on the DepEd K-12 framework — no manual input needed.  · DepEd Order No. 8, s. 2015

How grade computation works:
Each subject's grading weight is automatically determined by its subject type (Language / Science & Math / Skills). Scores from Quiz + Monthly Exam go to Written Works (WW), Activity + Participation + Attendance go to Performance Tasks (PT), and Periodical Exam scores go to Quarterly Assessment (QA).
